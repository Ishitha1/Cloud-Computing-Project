# ShareX - Ride Sharing Platform

A modern, responsive, and user-friendly ride-sharing web application built with a serverless architecture on AWS.

![ShareX Banner](https://via.placeholder.com/800x300?text=ShareX+Ride+Sharing)  
*(Replace with actual screenshot)*

## ✨ Features

- ✅ User Registration & Login (Amazon Cognito)
- ✅ Ride Booking System
- ✅ Interactive Map Integration
- ✅ AI Chatbot (Amazon Lex)
- ✅ Responsive Design (Mobile + Desktop)
- ✅ Serverless Backend (AWS Lambda + API Gateway)
- ✅ Secure Authentication with JWT
- ✅ Real-time UI Updates
- ✅ Automated CI/CD with AWS Amplify

## 🛠 Tech Stack

### Frontend
- HTML5, CSS3, Tailwind CSS
- JavaScript (ES6)
- Font Awesome Icons

### Backend & Cloud (AWS)
- **AWS Amplify** – Frontend Hosting & CI/CD
- **AWS Lambda** – Serverless Backend Logic
- **Amazon API Gateway** – REST API Management
- **Amazon Cognito** – Authentication & Authorization
- **Amazon DynamoDB** – NoSQL Database
- **Amazon Lex** – Conversational AI Chatbot
- **AWS IAM** – Security & Access Control
- **Amazon CloudWatch** – Monitoring & Logging

### Tools
- GitHub + GitHub Secrets
- VS Code
